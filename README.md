# Weather App

A clean weather app that shows current conditions and a 7-day forecast for any city in the world.

**[Live Demo](https://ibrahim-alattar.github.io/weather-app/)**

---

## Features

- Current temperature, humidity, wind speed, and weather condition
- 7-day forecast with daily min/max temperatures
- °C / °F toggle
- Dynamic weather icons
- Dark / Light theme

## Built With

- JavaScript (ES Modules)
- Webpack
- [Visual Crossing Weather API](https://www.visualcrossing.com/)
- [Meteocons](https://bas.dev/work/meteocons) — animated SVG weather icons

## Run Locally

```bash
git clone https://github.com/Ibrahim-Alattar/weather-app.git
cd weather-app
npm install
npm run dev
```
